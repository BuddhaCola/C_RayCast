## **Usage:**
This is Makefile project\
Clone it
```
git clone https://github.com/BuddhaCola/Cub3D.git
```
cd into
```
cd Cub3D
```
and compile
```
make
```
then launch by entering\
"$ ./cub3D <map>.cub", replacing <map>.cub with a map file (repository includes two: wolf.cub and bobville.cub)
```
./cub3D wolf.cub
```
